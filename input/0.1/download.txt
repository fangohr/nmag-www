.. contents: Installation methods


Method A: compile everything from source
----------------------------------------

Download a `tar file <download/nmag-0.1-all.tar.gz>`__ to compile nmag
and (most) required other software from source. `Installation instructions <install/install_a.html>`__.



Method B: Debian package
------------------------

Download deb file directly from http://nmag.soton.ac.uk/debian/dists/stable/unofficial/binary-i386. `Installation instructions <install/debian.html>`__.


Method C: Knoppix CD
--------------------

Download the iso image from `this directory <http://nmag.soton.ac.uk/cd-images>`__. `Installation instructions <install/knoppix.html>`__




Not sure which installation method to use?
------------------------------------------

Read the `installation instruction overview <install/index.html>`__.
