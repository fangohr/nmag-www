.. contents: Installation methods


Method A: compile everything from source
----------------------------------------

Download a `tar file <download/nmag-0.1.tar.gz>`__ to compile nmag and (most)
required other software from source.



Method B: Debian package
------------------------

Download deb file directly from http://nmag.soton.ac.uk/debian/dists/stable/unofficial/binary-i386 (see also `instructions <install/debian.html>`__)


Method C: Knoppix CD
--------------------

Download the iso image from `this directory <http://nmag.soton.ac.uk/cd-images>`__.
