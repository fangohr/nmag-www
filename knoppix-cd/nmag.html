Welcome to the Nmag micromagnetism simulator

This is a bootable Linux live-CD (based on Klaus Knopper's KNOPPIX) which contains the Nmag micromagnetism simulator. By booting from this CD, it is possible to test drive Nmag without having to install any software on harddrive.

Please visit the Nmag web page at http://nmag.soton.ac.uk for news, software updates, and further information on the Nmag micromagnetism simulator.

The Nmag developers, nmag@soton.ac.uk
