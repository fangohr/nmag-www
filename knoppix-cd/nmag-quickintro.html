A quick introduction for the impatient

To run the first Nmag demo example,

  1. Click on the monitor icon at the bottom of the screen to open up a command line window.
  2. Enter the command cd Desktop/nmag/example1 to go into the directory containing the first example.
  3. Then, enter the command nsim sphere1.py to start the simulation.
  4. When finished, run mayavi --vtk sphere1.vtk to start the MayaVi Data Visualizer. In the pull-down menu, select Visualize -> Modules -> VelocityVector.
